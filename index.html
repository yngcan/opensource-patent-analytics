Welcome to the Open Source Patent Analytics Project.

Our aim is to build a guide and resources for open source analysis of patent data. At the moment our plans cover the following.

  1. Getting Patent Data
  2. Tidying Patent Data
  3. Analysis
  4. Network Visualisation
  5. Dashboards
  6. Geographic Mapping

We are just getting started and the repository will fill up in the next few months.

Patent Datasets

We have created some patent datasets using WIPO Patent Landscape Reports and they are now publicly available for download and use here via Google Drive.

You can also access the datasets via the Github Repository but please use the Raw button on each page or you will simply receive the html page with the link on it.

More datasets will be loaded in due course.

Databases

If you have stumbled across this site looking for free sources of patent data here are some resources to get you started.

  • espacenet
  • patenscope
  • free patents online
  • patent lens
  • United States Patent and Trademark Office Applications and Grants Databases
  • China to 2012 via the Google & USPTO collaboration
  • Google Patents
  • Google Bulk Download for USPTO data

The excellent Patent Information User Group has plenty of other resources in its Patent Databases List.

In the meantime, thanks for visiting, you are welcome to take a fork from the repository or check back later as we start to develop the resource.
